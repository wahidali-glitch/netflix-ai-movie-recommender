# 🎬 Netflix AI Movie Recommender

An AI-powered Movie Recommendation System built using **Python, Streamlit, Machine Learning, and TMDB API**.

This project recommends similar movies based on content similarity and displays beautiful movie posters, ratings, release dates, and descriptions in a Netflix-style UI.

# 🧠 Machine Learning Used

This project uses:

- Content-Based Recommendation System
- K-Nearest Neighbors (KNN)
- Cosine Similarity

The model recommends movies similar to the selected movie based on movie metadata.
